# Mathematical Operators in JavaScript

A demonstration of the basic arithmetic (mathematical) operators.

```javascript
var a = 5;
var z = 3;

console.log(a + z); // 8         (Addition)
console.log(a - z); // 2         (Subtraction)
console.log(a * z); // 15        (Multiplication)
console.log(a / z); // 1.666...  (Division)
console.log(a % z); // 2         (Modulus / Remainder)
console.log(a ** z); // 125      (Exponentiation / Power)
```

---

These operators are commonly used in math-related expressions and calculations.
